# Vuttr (Very UseFul Tools to Remember)

## Requirements

- Nodejs v18.16.0

## How to execute

### clone
- Using ssh:
```
git clone git@github.com:LuisDomiciano/vuttr-graphql-api.git
```

- Using http:
```
git clone  https://github.com/LuisDomiciano/vuttr-graphql-api.git
```

```
npm run dev
```

### Routes

http://localhost:3000/graphql
